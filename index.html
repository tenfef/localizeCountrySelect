Localize Country select.

When showing a dropdown box on a form you can improve your user experience by preselecting the users country based on their IP addresss. This is a jquery plugin that allows you to do just that.

Simply include the JS file

<script src="localizeCountrySelect.js"></script>

Then run the jquery extension on your select box

$('#country_select').countryLocalizeSelect();
